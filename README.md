# teste-basico-react
Projeto para testar conhecimento básico de projeto React

## Requisitos
- npm;
- node;

## Subir projeto
- `npm install`
- `npm start`

## Rodar teste
- `npm test`

## Objetivo
1. Compile o projeto, caso tenha erros de compilação, resolva eles.
2. Acessar a página inicial e modificar o texto para `Olá meu react`.
